\# SysPulse: Real-Time System Monitor \& Logger



SysPulse is a lightweight, real-time command-line system monitoring utility built in Python. It provides an elegant terminal interface to track CPU Core utilization, RAM allocation, disk storage capacity, and active process footprints. Additionally, it features an asynchronous background logger that tracks system metrics over time for performance debugging.



\## Features

\- \*\*Real-Time Terminal Dashboard:\*\* Renders interactive, block-style progress bars for system metrics.

\- \*\*Per-Core CPU Monitoring:\*\* Displays live usage for each individual CPU core.

\- \*\*Automatic Background Logging:\*\* Persists historical data with timestamps to a structured `system\_log.csv` file.

\- \*\*Process Memory Footprint Tracker:\*\* Dynamically tracks and ranks the top 5 most memory-intensive background processes.

\- \*\*CLI Customizability:\*\* Command-line argument parsing to adjust refresh intervals dynamically.
